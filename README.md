# 🌅 은퇴바라기

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)

대한민국 세법 기준으로 **근로소득 + 배당·이자소득의 세후 실수령액**을 계산하고, **미국·국내 ETF 배당 전략**별 예상 수익을 시뮬레이션하며, **CNN Fear & Greed Index**로 시장 심리를 확인할 수 있는 종합 도구입니다.

## 페이지 구성

### 📊 소득 계산기 (`index.html`)
- **근로소득 세금 계산** — 근로소득공제, 기본공제, 근로소득세액공제, 표준세액공제 반영
- **4대보험 계산** — 국민연금·건강보험·장기요양·고용보험 직원 부담분 (2026년 요율)
- **금융소득 과세** — 2,000만원 이하 분리과세(15.4%) / 초과 시 종합과세 비교과세 자동 선택
- **은퇴 모드** — 연봉 0원 입력 시 금융소득 전용 모드로 전환 (지역가입자 건강보험 적용)
- **실시간 계산** — 슬라이더 또는 직접 입력으로 즉시 결과 반영
- **과세표준 구간 하이라이트** — 종합과세 적용 시 비교과세 방식(전액합산·초과분합산)에 따른 실제 합산 과세표준 기준으로 구간 자동 표시
- **이미지로 내보내기** — 결과 및 세율표를 PNG로 저장

### 📈 ETF 배당 계산기 (`etf.html`)
- **총 투자금액 슬라이더** — 최대 15억원까지 입력
- **미국/한국 ETF 전환** — 국가 토글로 전략 목록 전환
- **미국 ETF 전략 5종** — 성장 중심 / 균형형 / 고배당 중심 / SPYI 고배당형 / QQQI 고배당형 (SCHD·JEPI·JEPQ·SPYI·QQQI 조합)
- **한국 ETF 전략 4종** — SCHD형+커버드콜 / 순수 월배당 / ISA 절세 / 국내+해외 혼합
- **직접 구성** — ETF 종목별 비율을 슬라이더로 자유롭게 설정하는 커스텀 포트폴리오
- **세전·세후 동시 표기** — 미국 원천징수 15% / 국내 배당소득세 15.4% 반영
- **ETF별 배당 기여 내역** — 비중·투자금·월 배당·수익률 상세 표시

### 📊 CNN Fear & Greed Index (`fear-greed.html`)
- **반원형 게이지** — 0(극도의 공포) ~ 100(극도의 탐욕)을 5단계 색상으로 시각화
- **과거 비교** — 전일 종가 · 1주 전 · 1개월 전 · 1년 전 점수
- **변동 추이 차트** — 1주 / 1개월 / 3개월 / 6개월 / 1년 / 전체 토글, 호버 툴팁, 기간 통계(평균·최저·최고)
- **5개 세부 지표** — 시장 모멘텀 · 시장 변동성(VIX) · 주가 강도 · 풋콜 비율 · 안전자산 수요
- **지표별 해석 가이드** — 각 지표가 무엇을 측정하고 점수를 어떻게 읽는지 설명
- **30일 미니 스파크라인** — 각 세부 지표의 최근 추세를 등급별 색상으로 표시
- **CORS 프록시 자동 fallback** — CNN API 직접 호출 실패 시 프록시 경유, localStorage 1시간 캐시

## 적용 세법 기준 (2026년)

| 항목 | 내용 |
|------|------|
| 금융소득 종합과세 기준 | 2,000만원 초과 |
| 국민연금 | 4.75% (월 상한 637만원) |
| 건강보험 | 3.595% |
| 장기요양보험 | 0.4724% |
| 고용보험 | 0.9% |
| 지방소득세 | 소득세의 10% |
| 미국 ETF 원천징수 | 15% |
| 국내 ETF 배당소득세 | 15.4% |

## 사용 방법

별도 빌드 없이 브라우저에서 바로 열면 됩니다.

```bash
# 로컬 서버로 실행 (선택)
npx serve .
```

## 파일 구조

```
├── index.html       # 소득 계산기 — UI 마크업 + 스타일
├── main.js          # 소득 계산기 — 세금 계산 로직 및 UI 렌더링
├── etf.html         # ETF 배당 계산기 — UI 마크업 + 스타일
├── etf.js           # ETF 배당 계산기 — 전략 계산 로직 및 UI 렌더링
├── fear-greed.html  # CNN Fear & Greed Index — UI 마크업 + 스타일
└── fear-greed.js    # CNN Fear & Greed Index — 데이터 페치 및 차트 렌더링
```

## 주의사항

본 계산기는 **참고용**입니다. 개인별 공제 항목(부양가족, 연금저축, 의료비 등)에 따라 실제 세액은 다를 수 있으며, ETF 배당 수익률은 최근 1년(TTM) 기준으로 향후 변동될 수 있습니다.
