# 🌅 은퇴바라기

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)

대한민국 세법 기준으로 **근로소득 + 배당·이자소득의 세후 실수령액**을 계산하는 단일 페이지 계산기입니다.

## 주요 기능

- **근로소득 세금 계산** — 근로소득공제, 기본공제, 근로소득세액공제, 표준세액공제 반영
- **4대보험 계산** — 국민연금·건강보험·장기요양·고용보험 직원 부담분 (2026년 요율)
- **금융소득 과세** — 2,000만원 이하 분리과세(15.4%) / 초과 시 종합과세 비교과세 자동 선택
- **은퇴 모드** — 연봉 0원 입력 시 금융소득 전용 모드로 전환 (지역가입자 건강보험 적용)
- **실시간 계산** — 슬라이더 또는 직접 입력으로 즉시 결과 반영
- **과세표준 구간 하이라이트** — 현재 해당 구간 자동 표시

## 적용 세법 기준 (2026년)

| 항목 | 내용 |
|------|------|
| 금융소득 종합과세 기준 | 2,000만원 초과 |
| 국민연금 | 4.75% (월 상한 637만원) |
| 건강보험 | 3.595% |
| 장기요양보험 | 0.4724% |
| 고용보험 | 0.9% |
| 지방소득세 | 소득세의 10% |

## 사용 방법

별도 빌드 없이 `index.html`을 브라우저에서 바로 열면 됩니다.

```bash
# 로컬 서버로 실행 (선택)
npx serve .
```

## 파일 구조

```
├── index.html   # UI 마크업 + 스타일
└── main.js      # 세금 계산 로직 및 UI 렌더링
```

## 주의사항

본 계산기는 **참고용**입니다. 개인별 공제 항목(부양가족, 연금저축, 의료비 등)에 따라 실제 세액은 다를 수 있습니다.
