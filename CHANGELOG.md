# Changelog

## [1.6.0] - 2026-06-17

### Added
- ETF 배당 수익률(TTM) 자동 갱신 스크립트 추가 (`scripts/update_yields.py`) — 해외 ETF는 yfinance, 국내 ETF는 pykrx 로 수익률 조회 후 `etf.js` 갱신
- 미국 ETF 실시간 수익률 조회 추가 — 페이지 로드 시 Yahoo Finance chart 엔드포인트(yfinance 동일 소스)를 브라우저에서 직접 호출하여 TTM 수익률 자동 반영, CORS 프록시 자동 fallback + localStorage 1시간 캐시

## [1.5.2] - 2026-06-17

### Added
- GPIQ(Goldman Sachs Nasdaq-100 Core Premium Income ETF) 미국 ETF 목록 추가 (수익률 ~9.4%)

### Changed
- 미국 ETF 6종 배당 수익률을 최신 TTM 기준으로 갱신 — SCHD 3.4→3.3% · JEPI 8.8→8.1% · JEPQ 10.5→10.1% · SPYI 12.0→11.8% · QQQI 13.5% (유지)
- SOL배당다우 수익률 3.5→3.8% 갱신
- ETF 목록(미국·국내)을 배당 수익률 오름차순으로 정렬

## [1.5.1] - 2026-05-27

### Fixed
- Fear & Greed 5개 세부 지표 스파크라인이 표시되지 않던 버그 수정 — 세부 지표 시계열이 별도 `_historical` 키가 아닌 지표 객체 내부 `data` 배열에 있는 구조를 반영
- 세부 지표 스파크라인을 0~100 고정 스케일 대신 원본 지표값의 min/max 기준 자동 스케일링으로 변경 (지표별 단위 차이 대응)

## [1.5.0] - 2026-05-22

### Added
- CNN Fear & Greed Index 페이지 추가 (`fear-greed.html` / `fear-greed.js`)
- 상단 네비게이션에 Fear & Greed 메뉴 추가
- 반원형 게이지로 현재 점수(0~100) 및 등급(Extreme Fear ~ Extreme Greed) 시각화
- 과거 비교 카드 (전일 종가 · 1주 전 · 1개월 전 · 1년 전)
- 변동 추이 시계열 차트 — 1주 / 1개월 / 3개월 / 6개월 / 1년 / 전체 토글
- 차트 호버 툴팁 (날짜·점수·등급 색상) 및 기간 통계(평균·최저·최고)
- 5개 세부 지표 카드 추가 — 시장 모멘텀 / 시장 변동성(VIX) / 주가 강도 / 풋콜 비율 / 안전자산 수요
- 각 세부 지표에 30일 미니 스파크라인 및 "무엇/해석" 설명 추가
- CNN 비공식 API 직접 호출 + CORS 프록시 자동 fallback + localStorage 1시간 캐시
